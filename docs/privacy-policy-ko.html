개인정보 처리방침

최종 수정일: 2026년 4월 4일

Docklings(이하 "본 앱")은 김진호가 개발한 macOS 데스크톱 컴패니언 애플리케이션입니다. 본 개인정보 처리방침은 Docklings 사용 시 귀하의 정보를 어떻게 처리하는지 설명합니다.

1. 수집하는 정보

본 앱은 다음과 같은 데이터를 수집 및 처리합니다:

  • 채팅 메시지: AI 컴패니언과 대화하기 위해 앱에 입력하는 텍스트 메시지입니다. 이 메시지는 응답 생성을 위해 Anthropic의 Claude API로 전송됩니다.
  • API 키: API 모드 사용 시, Anthropic API 키가 사용자의 기기에 로컬로 저장됩니다.

2. 정보 이용 목적

  • 채팅 메시지는 AI 대화 기능을 제공하기 위한 목적으로만 Anthropic의 Claude API에 전송됩니다.
  • 채팅 메시지를 별도의 서버에 저장하지 않습니다. 모든 대화 데이터는 실시간으로 처리되며 사용자의 로컬 기기에만 저장됩니다.
  • 사용자 데이터를 광고, 분석 또는 추적 목적으로 사용하지 않습니다.

3. 제3자 서비스

본 앱은 다음 제3자 서비스를 이용합니다:

  • Anthropic Claude API: AI 응답 생성을 위해 채팅 메시지가 Anthropic 서버로 전송됩니다. Anthropic의 데이터 처리는 Anthropic 개인정보 처리방침에 따릅니다.

4. 데이터 저장

  • 모든 앱 데이터(채팅 기록, 설정, API 키)는 사용자의 Mac에 로컬로 저장됩니다.
  • 별도의 서버를 운영하지 않으며, 서버에 데이터를 저장하지 않습니다.
  • 앱과 관련 데이터를 시스템에서 삭제하면 언제든지 모든 앱 데이터를 제거할 수 있습니다.

5. 데이터 공유

제3조에 명시된 경우(AI 응답 처리를 위해 Anthropic API로 채팅 메시지를 전송하는 경우)를 제외하고, 사용자의 개인정보를 제3자에게 판매, 공유 또는 이전하지 않습니다.

6. 아동 개인정보 보호

본 앱은 만 13세 미만의 아동을 대상으로 하지 않습니다. 아동의 개인정보를 의도적으로 수집하지 않습니다.

7. 처리방침 변경

본 개인정보 처리방침은 수시로 업데이트될 수 있습니다. 변경 사항은 본 페이지에 수정 날짜와 함께 반영됩니다.
