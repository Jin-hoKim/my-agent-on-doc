Docklings User Guide

Everything you need to know to get started with your AI dock companion.

Getting Started

1 Launch the App

After installing Docklings from the Mac App Store, launch it from your Applications folder. The app runs as a menu bar application — you'll see a small icon appear in your menu bar.

Tip: Docklings runs in the background and doesn't appear in the Dock app switcher. Look for the icon in your menu bar (top-right of your screen).

2 Set Up Your Connection

Click the menu bar icon and open Settings. You have two ways to connect to Claude:

Mode Description Requirements
Claude CLI Uses your local Claude Code installation Claude Code CLI installed
Claude API Connects directly to Anthropic's API API key from console.anthropic.com

For API mode, enter your Anthropic API key in the settings. The key is stored securely on your device and never sent anywhere except Anthropic's servers.

3 Meet Your Companion

Once connected, an animated character will appear above your Dock. This is your AI companion, ready to chat anytime!

Chatting with Your Companion

Starting a Conversation

  1. Click the character above your Dock to open the chat window
  2. Type your message in the text field at the bottom
  3. Press Enter or click Send to submit
  4. Watch your companion react with expressions as Claude responds in real-time

Chat Features

  • Real-time streaming: Responses appear token-by-token as Claude generates them
  • Conversation memory: Claude remembers context within each conversation
  • New conversation: Click the new chat button to start fresh
  • Chat history: Access previous conversations from the menu bar

Settings

Connection

  • Connection Mode: Switch between Claude CLI and API mode
  • API Key: Enter your Anthropic API key (API mode only)
  • Claude Model: Choose between Haiku (fast), Sonnet (balanced), or Opus (most capable)

Appearance

  • Character: Select your preferred companion
  • Character Size: Adjust from 40pt (small) to 120pt (large) using the slider

Voice

  • Text-to-Speech: Toggle on/off to have responses read aloud
  • Voice Type: Choose from Male, Female, or Robot voice
  • Test: Preview the selected voice with a sample phrase

Menu Bar

Click the menu bar icon to access:

  • Connection status: Shows whether Claude is connected
  • Recent conversations: Quick access to your last 5 chats
  • Settings: Open the settings panel
  • Quit: Exit Docklings

Claude Models

Model Speed Capability Best For
Haiku 4.5 Fastest Good Quick questions, simple tasks
Sonnet 4.6 Fast Great Most conversations, balanced
Opus 4.6 Moderate Best Complex reasoning, detailed analysis
Tip: Start with Sonnet for everyday use. Switch to Haiku for quick answers or Opus when you need the most thoughtful responses.

Troubleshooting

Character not appearing above the Dock

  • Make sure Docklings is running (check for the menu bar icon)
  • Try quitting and relaunching the app
  • Check that your connection is set up in Settings

API connection errors

  • Verify your API key is correct in Settings
  • Check your internet connection
  • Ensure you have API credits at console.anthropic.com

"Rate limit exceeded" error

  • Wait a moment before sending another message
  • Consider switching to a different model
  • Check your API usage limits at Anthropic's console
