Docklings 사용 가이드

AI Dock 컴패니언을 시작하기 위한 모든 안내입니다.

시작하기

1 앱 실행

Mac App Store에서 Docklings를 설치한 후, 응용 프로그램 폴더에서 실행하세요. 메뉴바 앱으로 동작하며, 메뉴바에 작은 아이콘이 나타납니다.

팁: Docklings는 백그라운드에서 실행되며 Dock의 앱 전환기에는 표시되지 않습니다. 화면 우측 상단 메뉴바에서 아이콘을 확인하세요.

2 연결 설정

메뉴바 아이콘을 클릭한 후 설정을 열어주세요. Claude에 연결하는 두 가지 방법이 있습니다:

모드 설명 필요 사항
Claude CLI 로컬에 설치된 Claude Code를 사용 Claude Code CLI 설치
Claude API Anthropic API에 직접 연결 console.anthropic.com에서 API 키 발급

API 모드의 경우, 설정에서 Anthropic API 키를 입력하세요. 키는 사용자 기기에 안전하게 저장되며 Anthropic 서버 외에는 어디에도 전송되지 않습니다.

3 컴패니언 만나기

연결이 완료되면 Dock 위에 애니메이션 캐릭터가 나타납니다. 언제든 대화할 준비가 된 AI 컴패니언입니다!

컴패니언과 대화하기

대화 시작

  1. Dock 위의 캐릭터를 클릭하여 채팅 창을 엽니다
  2. 하단 입력란에 메시지를 입력합니다
  3. Enter 키를 누르거나 전송 버튼을 클릭합니다
  4. Claude가 실시간으로 응답하면서 캐릭터가 다양한 표정으로 반응하는 것을 감상하세요

채팅 기능

  • 실시간 스트리밍: Claude가 생성하는 응답이 토큰 단위로 실시간 표시됩니다
  • 대화 기억: Claude는 각 대화 내에서 이전 맥락을 기억합니다
  • 새 대화: 새 채팅 버튼을 클릭하여 새로운 대화를 시작합니다
  • 대화 기록: 메뉴바에서 이전 대화에 빠르게 접근할 수 있습니다

캐릭터 표정

컴패니언은 대화 상황에 따라 반응하는 20가지 이상의 애니메이션 표정을 가지고 있습니다:

표정 표시 시점
대기메시지를 기다리는 중
생각 중메시지를 처리하는 중
타이핑응답을 생성하는 중
신남길고 열정적인 응답
만족일반적이고 친근한 응답
웃음응답에 유머 키워드가 포함된 경우
하트 눈응답에 애정 표현 키워드가 포함된 경우
놀람예상치 못한 에러 발생
슬픔인증 오류 (잘못된 API 키)
화남요청 제한 초과
졸음5분 이상 대기 상태
음성 모드TTS 음성 재생 중

캐릭터

각각 고유한 개성을 가진 두 가지 캐릭터 중에서 선택하세요:

캐릭터 색상 성격
Nova 파랑 테크 감성, 분석적, 도움이 되는
Sprout 초록 밝고 활발한, 에너지 넘치는, 친근한

설정

연결

  • 연결 모드: Claude CLI와 API 모드 간 전환
  • API 키: Anthropic API 키 입력 (API 모드 전용)
  • Claude 모델: Haiku (빠름), Sonnet (균형), Opus (최고 성능) 중 선택

외관

  • 캐릭터: 원하는 컴패니언 선택
  • 캐릭터 크기: 슬라이더로 40pt(작게)부터 120pt(크게)까지 조절

음성

  • 음성 읽기(TTS): 켜기/끄기로 응답을 음성으로 읽어주기
  • 음성 종류: 남성, 여성, 로봇 음성 중 선택
  • 테스트: 선택한 음성을 샘플 문구로 미리 듣기

메뉴바

메뉴바 아이콘을 클릭하면 다음에 접근할 수 있습니다:

  • 연결 상태: Claude 연결 여부 표시
  • 최근 대화: 최근 5개 대화에 빠른 접근
  • 설정: 설정 패널 열기
  • 종료: Docklings 종료

Claude 모델

모델 속도 성능 추천 용도
Haiku 4.5 가장 빠름 좋음 빠른 질문, 간단한 작업
Sonnet 4.6 빠름 우수 일반적인 대화, 균형 잡힌 성능
Opus 4.6 보통 최고 복잡한 추론, 심층 분석
팁: 일상적인 사용에는 Sonnet을 추천합니다. 빠른 답변이 필요하면 Haiku로, 가장 심도 있는 응답이 필요하면 Opus로 전환하세요.

문제 해결

Dock 위에 캐릭터가 나타나지 않을 때

  • Docklings가 실행 중인지 확인하세요 (메뉴바 아이콘 확인)
  • 앱을 종료하고 다시 실행해 보세요
  • 설정에서 연결이 올바르게 되어있는지 확인하세요

API 연결 오류

  • 설정에서 API 키가 올바른지 확인하세요
  • 인터넷 연결 상태를 확인하세요
  • console.anthropic.com에서 API 크레딧이 있는지 확인하세요

"요청 제한 초과" 오류

  • 잠시 기다린 후 다시 메시지를 보내세요
  • 다른 모델로 전환을 고려하세요
  • Anthropic 콘솔에서 API 사용량 제한을 확인하세요

개인정보 및 데이터

  • 모든 데이터는 Mac에 로컬로 저장됩니다
  • 채팅 메시지는 Anthropic의 Claude API에만 전송됩니다
  • 분석, 추적, 제3자 데이터 공유가 없습니다
  • 전체 개인정보 처리방침 확인
